# Imperative vs Declarative in React

## Takeaways
- Imperative: specify *how* to build/update the UI step by step (e.g. manual DOM operations).
- Declarative: specify *what* the UI should look like for a given state; React handles DOM updates.

## Declarative React example
```jsx
function Counter() {
  const [count, setCount] = React.useState(0);
  return (
    <button onClick={() => setCount(c => c + 1)}>
      Count: {count}
    </button>
  );
}
```
