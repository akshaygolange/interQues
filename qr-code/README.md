# QR Code Generator

A React-based QR Code Generator that allows users to generate, copy, download, and clear QR codes.

## Features

* Generate QR codes from text or URLs
* Copy generated text to clipboard
* Download QR code as SVG
* Clear input and generated QR code
* Generate QR code using Enter key support
* Responsive and clean UI

## Tech Stack

* React.js
* react-qr-code
* JavaScript
* CSS

## Concepts Practiced

* useState
* useRef
* Controlled Components
* Event Handling
* Conditional Rendering
* Clipboard API
* Blob API
* File Download Functionality
